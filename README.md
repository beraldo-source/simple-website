## 🧭 Proposta do site

Plataforma voltada para **denúncias ambientais**, com recursos de:

- 📍 Localização de denúncias em mapa
- 📸 Upload de imagens e gifs
- 🕵️ Acompanhamento e fiscalização por um Administrador
- 👀 Visualização pública das denúncias em tempo real

O Administrador pode verificar o status de cada denúncia e **removê-la do banco de dados** caso ela tenha sido solucionada.

> ⚠️ As credenciais de login do Administrador devem ser inseridas manualmente no banco de dados.

## ⚙️ Como usar

1. **Instale as dependências:**
   Tenha o python e o MySql baixado (https://www.youtube.com/watch?v=AaISTiooIVU)
   abra o terminal e digite,
   
   pip install flask flask-mysqldb

   Depois de ter instalado todas as dependências, digite python app.py, ira mostrar o seu ip local, basta abrir ele no navegador.
   Não se esqueça de ter seu SQL aberto, com as colunas criadas e com as credenciais de acordo com o config.py
